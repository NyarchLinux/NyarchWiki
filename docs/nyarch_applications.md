# Nyarch Applications
This article contains the list of Nyarch Applications.
Some applications are only available for some Nyarch Versions.

## Catgirl Downloader
Catgirl downloader downloads random images of catgirls from multiple sources.

Features:
- Auto Reload
- Nya
- Multiple sources

Supports the following sources:

- [nekos.moe](https://nekos.moe)
- [waifu.im](https://waifu.im)
- [danbooru](https://danbooru.donmai.us)

##### Gnome Version

- **App ID**: `moe.nyarchlinux.catgirldownloader`
- **Launch Command**: `flatpak run moe.nyarchlinux.catgirldownloader`
- **Github Repository**: [https://github.com/NyarchLinux/CatgirlDownloader](https://github.com/NyarchLinux/CatgirlDownloader)
- **Language**: `Python`
- **Toolkit**: `GTK4 + Adwaita`

##### KDE Version

- **App ID**: `moe.nyarchlinux.catgirldownloaderqt`
- **Launch Command**: `catgirldownloaderqt`
- **Github Repository**: [https://github.com/NyarchLinux/CatgirlDownloaderQT](https://github.com/NyarchLinux/CatgirlDownloaderQT)
- **Language**: `Python`
- **Toolkit**: `QT + Kirigami (PySide6)`

## Nyarch Tour
Nyarch Tour launches automatically after first Nyarch installation. Its purpose is to guide the user to Nyarch features and how to use its desktop.

##### Gnome Version

- **App ID**: `moe.nyarchlinux.tour`
- **Launch Command**: `flatpak run moe.nyarchlinux.tour`
- **Github Repository**: [https://github.com/NyarchLinux/NyarchTour](https://github.com/NyarchLinux/NyarchTour)
- **Language**: `Python`
- **Toolkit**: `GTK4 + Adwaita`


##### KDE Version

- **App ID**: `moe.nyarchlinux.tourqt`
- **Launch Command**: `nyarchtourqt`
- **Github Repository**: [https://github.com/NyarchLinux/NyarchTourQT](https://github.com/NyarchLinux/NyarchTourQT)
- **Language**: `Python`
- **Toolkit**: `QT + Kirigami (PySide6)`
